# Loan Recovery Prediction – Full Stack Web App

A machine learning web application that predicts whether a loan applicant will default or repay, based on historical data.  
The model was trained on the Kaggle Loan Prediction Dataset using scikit‑learn.

## Features

- **Predict** default probability for new applicants.
- **Risk categorisation** – Low, Medium, High.
- **Interactive UI** built with Flask + HTML/CSS/JS.
- **Model persistence** – trained model loaded via `joblib`.

## Tech Stack

- **Backend:** Flask
- **Frontend:** HTML, CSS, JavaScript
- **ML Libraries:** scikit‑learn, pandas, numpy
- **Deployment ready** (can be hosted on Render, PythonAnywhere, etc.)
